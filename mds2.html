Exercício Br modelo

O modelo conceitual de banco de dados é uma representação abstrata e de alto nível da estrutura de dados de um sistema. Ele descreve os principais entidades, atributos e relacionamentos de forma independente de tecnologias específicas, servindo como uma ponte entre os requisitos do usuário e o projeto lógico do banco de dados. Geralmente é criado com a ajuda de diagramas, como o modelo Entidade-Relacionamento (ER), para facilitar a visualização e o entendimento do sistema por desenvolvedores e stakeholders.
